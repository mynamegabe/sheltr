# hackrift25

## deployment
1. `docker-compose up --build`

## frontend

1. `bun create vite@latest`
2. `bun add tailwindcss @tailwindcss/vite`
3. `bun add -D @types/node`
4. `bunx --bun shadcn@latest init`
5. `bunx --bun shadcn@latest add button`
6. `bun add @tanstack/router-plugin`
7. `bun add @tanstack/react-router`

### references
1. https://ui.shadcn.com/docs/installation/vite
2. https://tanstack.com/router/v1/docs/framework/react/installation/with-vite
